# 🛒 CompraX - Sua Compra Inteligente em Família

O **CompraX** é um ecossistema mobile projetado para otimizar e coordenar as compras domésticas em tempo real. Desenvolvido com uma arquitetura escalável de SaaS, o app resolve o problema de falta de comunicação familiar no mercado, permitindo sincronização instantânea entre múltiplos dispositivos.

---

## 🚀 Funcionalidades Principais

- 🔐 **Autenticação Segura:** Sistema de login e recuperação de senha via Firebase Auth.
- 👨‍👩‍👧‍👦 **Lógica de Família (Multi-tenancy):** Criação de grupos colaborativos via código de convite único. Cada família possui seu ecossistema isolado de dados.
- 🛡️ **Privacidade Seletiva:** O proprietário da lista decide quais listas são privadas (pessoais) e quais são compartilhadas com o grupo através de um sistema de "Portal Cloud".
- ☁️ **Sincronização Real-time:** Utilização do Firestore para atualizações instantâneas. Se um membro marcar um item, todos os outros visualizam a mudança no exato segundo.
- 📍 **Exploração de Atacadistas:** Integração com **Google Places API** para localizar mercados próximos baseado na geolocalização (GPS) do usuário.
- 🚗 **Navegação Inteligente:** Deep Linking nativo com Google Maps e Waze para traçar rotas automáticas para o mercado escolhido.
- 📊 **UX Avançada:** Barra de progresso dinâmica, ordenação inteligente de itens e interface otimizada para uso com teclado mobile.

---

## 🛠️ Stack Tecnológica

- **Frontend:** React Native (Expo Router)
- **Linguagem:** TypeScript
- **Backend:** Firebase Firestore (NoSQL)
- **Auth:** Firebase Authentication
- **APIs Externas:** Google Maps Platform (Places API)
- **Segurança:** Gestão de ambiente com Variáveis de Ambiente (.env)
